== Description ==
Add image hover system :

`[imghover]<img src="image1.jpg"><img src="image_hover.jpg">[/imghover]`

The first image is replaces by the second when hover

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/mousehover` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
